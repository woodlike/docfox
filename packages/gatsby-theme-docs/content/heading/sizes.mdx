# Headings

## Heading Sizes

Covers sizes from **s** to **xxxl**.
The heading component also has a `tag` prop that accepts all six section levels according the spec.

### Examples

<Heading as="h1" size="xxl">
  Healing Flowers Collection
</Heading>
<Heading as="h2" size="xl">
  Healing Flowers Collection
</Heading>
<Heading as="h3" size="l">
  Healing Flowers Collection
</Heading>
<Heading as="h4" size="m">
  Healing Flowers Collection
</Heading>
<Heading as="h5" size="s">
  Healing Flowers Collection
</Heading>
<Heading as="h5" size="xs">
  Healing Flowers Collection
</Heading>
